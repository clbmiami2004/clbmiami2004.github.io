# Christian Lorenzo GitHub Pages Site

Static GitHub Pages site for app landing pages, privacy policies, and professional work.

Live URL after publishing:

https://clbmiami2004.github.io

## Structure

- `index.html` - professional homepage
- `support.html` - shared support and contact page
- `apps/my-meeting-transcriber.html` - app landing page
- `apps/im-in-danger.html` - app landing page
- `privacy/my-meeting-transcriber.html` - privacy policy for My Meeting Transcriber
- `privacy/im-in-danger.html` - privacy policy for I'm in Danger
- `assets/styles.css` - shared styling

## Updating

Edit the HTML files, then commit and push to `main`.
